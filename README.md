# 🚢 Titanic Passenger Analysis Dashboard

An interactive dashboard built using **Python** and **Streamlit** to analyze the famous Titanic passenger dataset.

The project demonstrates how to perform data analysis, passenger filtering, statistical calculations, and interactive data visualization using one of the most popular datasets available in Seaborn.

## 🌐 Live Demo
https://h5ygkeqcdvtxeekr5icwmy.streamlit.app/

---

# 📊 Dataset

**Dataset Name:**

Titanic Dataset (Seaborn)

The dataset contains passenger information such as:

- Passenger Class
- Survival Status
- Name
- Sex
- Age
- Number of Siblings/Spouses
- Number of Parents/Children
- Fare
- Embarkation Port
- Family Size
